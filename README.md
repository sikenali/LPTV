# LPTV - 本地 IPTV 播放器

一个简洁优雅的本地 IPTV 播放器，支持 M3U/M3U8 直播源导入与播放，提供频道管理、收藏、定时更新等实用功能。

## 功能特点

### 📺 核心功能

- **多播放器内核支持**：HLS.js、原生 Video、DPlayer 三种播放器内核，适应不同播放场景
- **直播源管理**：支持 URL 和本地文件导入 M3U/M3U8 直播源
- **频道分类**：支持全部、央视频道、卫视频道、其他频道等分类
- **频道收藏**：一键收藏喜欢的频道，方便快速访问
- **定时更新**：自动定期更新直播源，确保频道列表始终最新
- **多主题支持**：深色、蓝色、纯黑、浅色四种主题模式
- **界面自定义**：支持主题色、字体大小、透明效果、圆角弧度等自定义
- **播放设置**：支持默认画质、解码方式、播放行为等设置

### 🎨 界面设计

- **响应式布局**：适配桌面端和移动端
- **现代化 UI**：简洁优雅的界面设计
- **实时预览**：主题切换实时预览效果
- **流畅动画**：平滑的过渡和交互动画

### 🔧 技术特点

- **本地存储**：使用 SQL.js (SQLite) 本地存储频道和设置
- **性能优化**：频道去重、缓存机制，提升加载速度
- **错误处理**：完善的错误处理和用户提示
- **测试覆盖**：单元测试确保功能稳定性

## 技术栈

- **前端框架**：Vue 3 + TypeScript + Vite
- **状态管理**：Pinia
- **路由**：Vue Router
- **样式**：SCSS
- **播放器**：HLS.js、DPlayer
- **数据库**：SQL.js (SQLite)
- **图标**：Remix Icon

## 快速开始

- Node.js 16.0 或更高版本

1. 克隆项目

   ```bash
   npm run build
   ```

   ```bash

   ```

## 使用指南

### 1. 导入直播源

#### URL 导入

1. 进入「源管理」页面
2. 选择频道分类（全部、央视频道、卫视频道、其他频道）
3. 输入 M3U/M3U8 直播源 URL
4. 点击「导入」按钮

#### 本地文件导入

1. 进入「源管理」页面
2. 点击「添加URL地址」按钮
3. 在弹出的模态框中选择「本地文件」标签
4. 拖拽文件到上传区域或点击选择文件
5. 输入源名称（可选）
6. 点击「确定」按钮

### 2. 频道管理

- **切换源**：点击源列表中的源名称切换当前活动源
- **编辑源**：点击源列表中的「编辑」按钮修改源信息
- **删除源**：点击源列表中的「删除」按钮删除源
- **更新源**：点击源列表中的「更新」按钮手动更新源

### 3. 频道播放

- **点击频道**：在频道列表中点击频道开始播放
- **全屏播放**：点击播放器中的全屏按钮进入全屏模式
- **音量控制**：点击播放器中的音量按钮调整音量
- **播放器内核**：在「播放设置」中选择合适的播放器内核

### 4. 频道收藏

- **收藏频道**：在频道列表中点击频道右侧的星标按钮
- **查看收藏**：点击顶部导航栏中的「收藏」标签
- **取消收藏**：在收藏列表中点击频道右侧的星标按钮

### 5. 定时更新

- **开启全局自动更新**：在「定时管理」页面开启「全局自动更新」
- **设置更新频率**：选择每小时、每6小时、每12小时、每天或每周
- **设置更新时间**：设置每天自动更新的具体时间
- **设置重试次数**：设置更新失败时的自动重试次数

### 6. 界面设置

- **主题切换**：在「界面设置」页面选择深色、蓝色、纯黑或浅色主题
- **主题色**：选择应用的主题强调色
- **字体大小**：选择小、中、大三种字体大小
- **透明效果**：开启或关闭界面元素半透明毛玻璃效果
- **圆角弧度**：调整界面元素的圆角弧度

## 常见问题

### 1. 播放出现卡顿或花屏

- 尝试切换到软解码模式
- 尝试降低播放画质
- 检查网络连接

### 2. 直播源无法导入

- 检查 URL 是否正确
- 检查网络连接
- 检查文件格式是否为 M3U/M3U8

### 3. 定时更新失败

- 检查网络连接
- 检查直播源 URL 是否可访问
- 查看更新历史记录了解失败原因
