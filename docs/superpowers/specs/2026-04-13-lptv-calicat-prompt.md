# LPTV 电视直播播放器 - Calicat 设计 Prompt

## 项目概述

设计一款现代简洁的电视直播播放类 Web 应用界面 LPTV，采用深色主题，布局规整清晰，适配电脑端/电视端交互。整体分为**顶部导航栏、左侧功能菜单/列表区、右侧主内容区**三大区域，层次分明，突出播放内容与操作便捷性。

---

## 全局规范

### 色彩体系
- 背景主色：`#141414`（深黑）
- 背景次色：`#1f1f1f`（深灰，侧边栏/面板背景）
- 卡片/面板：`#2a2a2a`（中灰）
- 边框/分割线：`#333333`
- 主要文字：`#ffffff`（纯白）
- 次要文字：`#999999`（浅灰）
- 品牌主色：`#3b82f6`（亮蓝，按钮/选中态/Logo）
- 高亮/悬停：`#60a5fa`（亮蓝浅）
- 收藏爱心：`#ef4444`（红色，实心/空心切换）
- 成功提示：`#22c55e`
- 错误提示：`#ef4444`

### 排版规范
- 标题：20px / bold → Logo、页面标题
- 副标题：16px / semibold → 分组标题、卡片标题
- 正文：14px / normal → 列表项、描述文字
- 辅助：12px / normal → 时间戳、状态标签

### 圆角与阴影
- 卡片/按钮圆角：6-8px
- 卡片 hover 阴影：`0 4px 12px rgba(0,0,0,0.3)`
- 弹窗阴影：`0 8px 32px rgba(0,0,0,0.4)`

### 交互动效
- 颜色过渡：200ms ease
- 展开/折叠：300ms ease（CSS max-height 过渡）
- 删除动画：300ms 缩小淡出

## 设计细节补充

### 组件状态
- **按钮**：
  - 默认：背景 `#3b82f6`，白色文字
  - hover：背景 `#60a5fa`
  - 禁用：背景 `#555`，文字 `#999`
- **输入框**：
  - 默认：背景 `#1f1f1f`，白色文字，边框 `#333`
  - focus：边框 `#3b82f6`
  - 错误：边框 `#ef4444`
- **列表项**：
  - 默认：透明背景
  - hover：背景 `#2a2a2a` 或 `#333333`
  - 选中/当前：背景 `#2a2a2a`，左侧蓝色竖线标记

### 空状态/加载状态
- **骨架屏**：灰色矩形块闪烁动画（频道列表加载时）
- **空列表**：居中图标 + 提示文字（灰色调）
- **错误状态**：红色图标 + 错误描述 + 重试按钮

### 响应式断点（可选）
- 桌面端（≥1280px）：完整三栏布局
- 平板端（768px-1279px）：左侧边栏宽度缩减至 240px
- 移动端（<768px）：暂不设计（当前项目以电脑端/电视端为主）

## 设计输出要求

1. **页面数量**：3 个主页面（频道页、收藏页、设置页）+ 1 个弹窗（导入源）
2. **页面尺寸**：1920×1080（桌面端标准分辨率）
3. **组件库**：使用现代简约风格组件，线性图标
4. **原型交互**：
   - 导航栏三个按钮可点击切换页面
   - 频道分组标题可展开/折叠（箭头旋转动画）
   - 爱心收藏图标可切换空心/实心
   - 导入源弹窗 Tab 切换
   - 设置页左侧子菜单切换
